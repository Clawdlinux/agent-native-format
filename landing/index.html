ACL — Agent Context Language

A compact, machine-native representation of structured data, designed for LLM agents instead of humans.

Every system AI agents consume — Kubernetes APIs, OpenAPI specs, database schemas — was designed for human eyes. JSON braces, field descriptions, vendor extensions, repeated key names: 80–95% of the bytes don't change the agent's decision. They burn the context window.

ACL is what those bytes look like when you redesign them for the consumer. Hand-written translators map any structured source (~250 lines of Go each) into a line-oriented format with no JSON ceremony. The compression isn't gzip — it's deliberate field selection plus a wire format LLMs tokenize cheaply.

Try it in 30 seconds

git clone https://github.com/Clawdlinux/ninevigil-acp
cd ninevigil-acp
make build-acl

# Compress a real OpenAPI spec
bin/acl tokens pkg/aclhttp/testdata/petstore.json
# tokens:   1492  (cl100k_base)

bin/acl encode openapi pkg/aclhttp/testdata/petstore.json | bin/acl tokens -
# tokens:   201   (cl100k_base)

Three independent paths in docs/quickstart.md: CLI (30s), Go library (2min), Python decoder (1min).

Use it from your stack

CLI (any language)

# Pipe stdin/stdout
cat my-spec.json   | acl encode openapi - | acl tokens -
cat schema.sql     | acl encode pg      - | acl tokens -

# Decode an ACL document into JSON for inspection
acl decode state.acl --indent 2

Python (drop into any agent runtime)

import acp_acl

doc = acp_acl.decode(open("state.acl").read())
print(doc.directives["ns"])             # 'payments'
print(doc.section("pods").summary)      # '5/5 ok'
for row in doc.section("pods").rows:
    print(row.id, row.fields, row.flags)

# Optional: tokenizer-accurate measurement
print(acp_acl.count_tokens(open("state.acl").read()))   # 260

Go

import "github.com/Clawdlinux/ninevigil-acp/pkg/aclhttp"

spec, _ := os.ReadFile("petstore.json")
out, _ := aclhttp.Encode(spec)            // OpenAPI -> ACL bytes

What ships today

  • ACL v0.1 wire-format spec — frozen, CC BY 4.0 (docs/acl-spec.md)
  • Go reference encoder/decoder — round-trip stable, fuzzed, Apache 2.0
  • Three translators — Kubernetes (132× live), OpenAPI 3.x (68× on GitHub spec), Postgres schema (3.5× vs realistic pg_dump)
  • acl CLI — encode / decode / tokens / version, single 15 MB distroless container
  • Python acp-acl decoder — pure Python, zero deps, optional tiktoken extra
  • 540/1620-trial agent-accuracy benchmark — every fixture and prompt committed; reproducible for $1
  • ACP server — HTTP API that returns ACL-shaped Execution Manifests on top of MCP (the protocol delivery layer for ACL)
