# Licensing

The ACP PoC intentionally uses a split license strategy.

## Runtime

The commercial runtime is licensed under Business Source License 1.1.

Applies to:

- `cmd/`
- `internal/`
- `deploy/` runtime manifests unless otherwise noted

The Change Date is **2029-05-02**. On that date, the runtime converts to
Apache License, Version 2.0.

## Protocol spec

The protocol specification is intended to become public and ecosystem-friendly.

Applies to:

- `SPEC.md`
- `docs/protocol.md`

License: **CC BY 4.0**.

## SDKs and adapters

The client surfaces should be easy for developers to adopt.

Applies to:

- `pkg/`
- `adapters/`

License: **Apache License, Version 2.0**.

## Benchmark methodology

The benchmark methodology should be reproducible and inspectable.

Applies to:

- `benchmark/scenarios/`
- benchmark report templates
- methodology docs

License: **Apache License, Version 2.0** unless a file states otherwise.
